# Social Media Prompt Template

Use this prompt to generate social media content for Ikerd Law Firm.

---

## Prompt

```
Write social media content for Ikerd Law Firm (Lafayette, Louisiana) about [TOPIC].

Platform: [Facebook / Instagram / LinkedIn / X / specify]
Format: [single post / carousel / series of posts]

Follow all rules in CLAUDE.md. Specifically:
- No contractions, no em dashes, no cliche phrases
- No outcome guarantees
- Professional and compassionate tone, slightly more conversational for social
- Verify all Louisiana legal information for accuracy

Deliver the following:
1. Post text with appropriate length for the platform
2. Suggested hashtags (5-10 relevant hashtags)
3. Suggested image description or visual concept
4. Alt text for the suggested image
5. A call to action directing to ikerdlaw.com or to contact the firm
```

---

## Example Usage

```
Write social media content for Ikerd Law Firm (Lafayette, Louisiana) about the importance of hiring a lawyer after a car accident.

Platform: Facebook
Format: single post

Follow all rules in CLAUDE.md...
```
